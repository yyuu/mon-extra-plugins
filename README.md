# mon-extra-plugins

extra plugins for mon monitoring system.

### Installing mon-extra-plugins

    $ git clone git://github.com/yyuu/mon-extra-plugins.git
    $ cd mon-extra-plugins
    $ ./install.sh

This will install mon-extra-plugins into `/usr/local`. If you do not have
write permission to `/usr/local`, you will need to run `sudo
./install.sh` instead. You can install to a different prefix by
setting the `PREFIX` environment variable.
